# Contributing to Awesome AI Tools 2026

Thanks for your interest in contributing! This list aims to be the most useful, up-to-date directory of AI tools available.

## How to Add a Tool

1. **Fork** this repository
2. **Add** the tool to the correct category in `README.md`
3. **Format** your entry as a table row:
   ```
   | [Tool Name](https://official-url.com) | One-line description. | Free/Freemium/Paid | [Review](https://whatstrending.ai/tools/slug) |
   ```
4. **Submit** a Pull Request with the tool name as the PR title

## Guidelines

- **One tool per PR** to keep reviews simple
- **Tool must be publicly available** (no private betas or waitlists)
- **Description must be factual** — no marketing language, no superlatives like "best" or "revolutionary"
- **No affiliate links** — official URLs only
- **No duplicates** — check the list first
- **Correct category** — if a tool fits multiple categories, pick the primary one
- **Pricing must be accurate** — Free (truly free), Freemium (free tier + paid), Paid (no free tier)

## How to Suggest a New Category

Open an Issue with:
- Category name
- Why it deserves its own section
- At least 3 tools that would go in it

## How to Report an Error

Open an Issue with:
- Which tool/entry has the error
- What the correct information is
- Source for the correction

## Code of Conduct

Be respectful. No spam. No self-promotion in issues or discussions. PRs that add genuine tools are always welcome.
